# Codex 自动视频剪辑工作流 Skill

这是一个可安装到 Codex 的 **Codex 自动视频剪辑工作流 Skill**，用来让 Codex 按固定底层逻辑自动剪辑口播视频，把原始口播素材处理成适合发布的短视频成片。

它保存了自动化剪辑、口播清理、重复话术删除、停顿删除、文案字幕对齐、BGM、音效、素材匹配、真实视频 cutaway、动态信息贴片、素材来源记录和最终 QA 的底层逻辑。以后换新窗口或给同事使用时，不需要重新复制一长串提示词。

## 核心使用方式

这个 Skill 的核心输入是 **已经录好的口播视频**。

推荐流程：

```text
1. 先用手机、相机、电脑摄像头或录屏软件录好口播视频。
2. 把原始口播视频放进输入文件夹。
3. 如果有正式文案，把文案用同名 .txt / .md / .docx 放在同一个输入文件夹。
4. 如果有自己的 BGM，把音乐放进 BGM 文件夹。
5. 在 Codex 里调用这个 Skill，让它按底层逻辑自动剪辑。
6. 成片、QA 报告、素材来源记录输出到输出文件夹。
```

最推荐的输入结构：

```text
input/
  视频名.mp4
  视频名.txt
bgm/
  轻快背景音乐.mp3
output/
```

## 适用场景

最适合：

- 真人口播视频自动剪辑；
- 财经、美股、商业、AI、技术、知识类解释视频；
- 有正式文案的口播视频；
- 需要删除停顿、重复、错话、重说的原始口播；
- 需要自动加字幕、BGM、轻音效和素材 cutaway 的短视频；
- 需要团队复用同一套剪辑底层逻辑的批量视频生产。

也适合：

- 多段口播视频按顺序合并，例如 `part1` + `part2`；
- 录屏教程加真人口播或圆形头像 PIP；
- 产品介绍、软件教程、工具演示、GitHub/网页/数据页面讲解；
- 访谈、课程切片、知识分享、创业复盘、商业案例讲解；
- 4:3 横板讲解、9:16 竖屏短视频、16:9 横屏教程。

需要另行扩展或不建议直接当默认流程：

- 没有口播、只有音乐和画面的 MV/混剪；
- 复杂剧情短片、广告片、电影级叙事剪辑；
- 需要大量人工审美决策的品牌大片；
- 没有文案、没有清晰讲话、音质很差的素材；
- 纯靠素材自动生成完整视频的场景。它可以辅助做，但当前最稳定的主流程仍是“口播视频自动剪辑”。

## 它能做什么

- 安装后先做环境检查和首次使用向导；
- 提醒用户选择 4:3、9:16、16:9 或保持源比例；
- 自动找到输入文件夹里的最新口播视频；
- 多段视频合并前做预检，防止把错误主题混进去；
- 先清理口播，再加素材；
- 删除停顿、重说、错话、半句话、重复观点；
- 使用用户提供的正式文案校准字幕和素材节奏；
- 默认清理后 1.2 倍语速，保持音高；
- 默认导出 4:3，1440x1080，30fps；
- 支持用户指定竖屏 9:16 或横屏 16:9；
- 默认加字幕、轻快 BGM、轻音效；
- 按句子级匹配寻找或生成素材；
- 真实视频、录屏、电影感动图、全屏 cutaway、动态信息贴片混合使用；
- 拒绝素材重复、弱匹配素材、难看素材、PPT 感太重的素材；
- 生成素材来源表、规则审计、抽帧 QA。

## 当前默认风格

```text
成片比例：首次使用先选择；未选择时才默认 4:3
4:3 分辨率：1440x1080
9:16 分辨率：1080x1920
16:9 分辨率：1920x1080
帧率：30fps
格式：MP4
语速：口播清理后 1.2 倍
字幕：默认内嵌
BGM：默认轻快、正向、低音量
音效：默认轻量、变化、不固定
```

2 分钟左右的财经、商业、技术解释视频，默认节奏：

```text
真人可见：约 45%-55%
素材主导：约 45%-55%
全屏真实或视频化 cutaway：通常 5-7 个
真人段落动态信息贴片：约 8-14 个
```

## 成片质量闭环

每次正式出片都要同时守住四条质量线：

```text
口播清理不丢：先删停顿、错话、重复、重说，再做视觉包装。
素材匹配不丢：素材必须贴当前句子，同时好看、清晰、无重复。
字幕/BGM/音效不丢：正式成片默认字幕对齐、BGM 轻快、音效轻且有变化。
QA 审计不丢：导出前检查比例、解码、音量、字幕、素材来源、抽帧/事件图。
```

如果新增风格导致其中一条消失，例如只有素材没有贴片、只有贴片没有真实 cutaway、字幕和声音不同步、BGM 压人声、素材重复或弱匹配，都不能当作正式成片交付。

## 不能妥协的规则

- 先清理口播，再加素材。
- 不能用 B-roll、音乐、音效掩盖口播重复。
- 同一个素材不能在一条视频里重复出现。
- 不能把弱素材包硬塞进成片。
- 不能把 PPT MP4 当成真实视频 cutaway。
- 不能用泛 AI、泛商业、泛财经素材硬凑。
- 素材必须同时好看且匹配当前句子。
- 正式成片默认要字幕、BGM、音效和 QA。
- 素材站/API 结果只是候选池，不能直接进时间线。
- 多段视频必须先检查，不允许混错源文件。

## 边界

这个仓库是 Codex Skill，不是独立视频编辑软件。它提供工作流、规则、检查脚本和决策逻辑。真正出片仍需要本机有输入/输出文件夹、FFmpeg、Python 包、可选转写工具、BGM、素材和渲染脚本。

它的第一定位是“口播视频自动剪辑工作流”，不是通用影视后期软件。没有口播或没有清晰人声的视频，需要先改成对应的工作流再处理。
